Reduces number of columns in a given dataset. It is possible to specify columns to leave in a dataset by specifying their names or indices or alternatively specify columns to remove from a dataset by specifying their names or indices.

Example:

Given original dataset containing records about orders made by clients:

firstName lastName orderDate address phoneNumber
Alice Smith 10.11.2015 New York, Harrison Street, 507 263-86-335
Bob Johnson 13.11.2015 Richmond, Main Street, 17 227-11-594
Alice Williams 25.11.2015 New York, East 52nd Street, 55 493-78-145
Mary Williams 29.11.2015 Florida, Orange, Orlando, East Winter Park Road, 1842 278-32-169

It is possible to leave only columns containing information about person's first name, last name and phone number. This can be done in several ways:

firstName lastName phoneNumber
Alice Smith 263-86-335
Bob Johnson 227-11-594
Alice Williams 493-78-145
Mary Williams 278-32-169

{{function.take?"Take only listed columns":"Take all columns EXCEPT of listed"}}
Specify column names Specify range of column indices
